# AI Resume Analyzer & Job Coach

An AI-powered resume analyzer that gives you:
- 📊 ATS Score (0-100)
- 🎯 Skill gap analysis with learning suggestions
- 🎤 5 tailored interview Q&As in STAR format
- ✏️ 3 bullet point rewrites with before/after
- 📋 Full coaching report + LinkedIn About draft

## Built With
- LLaMA 3.3 70B via Groq
- Streamlit
- FastAPI
- Python
